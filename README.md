# FEM3d
## 仓库介绍
本仓库为GAMES201大神的三维fem实例，相关代码注释已经写进代码文件
## 环境配置
运行环境为 Python3.8 需要0.7版本 taichi库
```bash
pip install taichi==0.7.13
```
## 运行方式
在命令行中 python fem-3d.py即可运行
```bash
python fem-3d.py
```
